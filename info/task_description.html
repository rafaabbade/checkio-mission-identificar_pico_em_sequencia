Enunciado

Marcos, um geólogo, está analisando dados de elevação de terrenos para identificar picos. Ele precisa de uma função que encontre a primeira ocorrência de um pico na sequência, onde um pico é definido como um elemento maior que seus vizinhos imediatos. Escreva uma função que receba uma lista de números inteiros e retorne o índice do primeiro pico encontrado. Se não houver picos, retorne -1.

Entrada: Uma lista de números inteiros.

Saída: O índice do primeiro pico encontrado ou -1 se não houver picos.
